# Mexico Toy Sales Dashboard — Power BI

## Project Overview
An advanced interactive Power BI dashboard analyzing toy retail sales performance
across products, store locations, and quarterly trends for Maven Toys Mexico.
Built with Bookmarks, Slicers, and dynamic filtering for enhanced interactivity.

## Dataset
- Retail sales data across multiple Maven Toys store locations in Mexico
- Covers quarterly performance across product categories and store types
- Store types: Residential, Downtown, Commercial, Airport

## Key Metrics
| KPI | Value |
|---|---|
| Total Units Sold | 10,90,565 |
| Total Sales | $1,44,44,572 |
| Total Profit | $40,14,029 |
| Total Cost | $1,04,30,543 |
| Profit Margin | 27.8% |

## Key Business Insights
- Q4 is peak sales quarter — driven by holiday season demand
- Ciudad de Mexico stores generate highest profit across all locations
- Colorbuds is the top profit-generating product
- Residential channel shows declining trend Q1 to Q4 — needs strategic review
- Overall profit margin of 27.8% is strong for toy retail segment

## Advanced Power BI Features Used
| Feature | Purpose |
|---|---|
| Bookmarks | Toggle between report views |
| Slicers | Filter by Year, Store, Product, Category |
| Combo Chart | Sales vs Profit quarterly comparison |
| Area Chart | Multi-channel sales trend over time |
| DAX Measures | Profit Margin, Total KPI calculations |

## Slicers / Filters Available
- Year
- Store Location
- Product Name
- Product Category

## Dashboard Pages
| Page | Content |
|---|---|
| Overview | KPI cards + quarterly trend |
| Product Analysis | Profit by product breakdown |
| Store Analysis | Profit and sales by store location |

## Tools Used
- Power BI Desktop
- DAX (calculated measures for KPIs and margins)
- Power Query (data cleaning and transformation)
- Bookmarks & Buttons (navigation and interactivity)
